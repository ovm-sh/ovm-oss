# Code of Conduct

## Our Standards

OVM is a small project and we aim for a small, high-signal community. Everyone is welcome — maintainers and contributors agree to:

- **Be kind.** Assume good intent. Disagree about ideas, not people.
- **Be direct.** Concise feedback helps everyone — but stay professional.
- **Stay on topic.** Issues and PRs are for OVM; general discussion goes elsewhere.
- **No harassment.** We don't tolerate discriminatory language, personal attacks, or dog-piling.

## Scope

This applies to GitHub issues, pull requests, commit messages, and any official OVM communication channels.

## Enforcement

Report violations via a private security advisory or email the maintainers. Responses range from a warning to removal depending on severity. Decisions are final.
